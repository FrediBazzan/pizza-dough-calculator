How to Use Dough Calculator

To use the Dough Calculator simply follow the steps below:

Select the pizza size

Choose between small size (20cm), medium size (25cm) or large size (30cm).

Select the nº of Pizzas

How many pizzas do you want to make? Select the number of pizzas for you recipe.

Get your Ingredients list

After calculate you will see the list of ingredients you need to prepare your pizzas.
